<action>
  Modify ReliquarySlot.vue to use Transition for expand/collapse body. Since height animation requires JavaScript for accurate sizing, use custom transition approach:
  
  1. Keep existing isOpen ref
  2. Add height calculation with ResizeObserver in script setup:
  
  ```typescript
  const bodyRef = ref<HTMLElement | null>(null);
  const computedHeight = ref('0px');
  
  function updateHeight() {
    if (!bodyRef.value) {
      computedHeight.value = '0px';
      return;
    }
    computedHeight.value = isOpen.value 
      ? `${bodyRef.value.scrollHeight}px` 
      : '0px';
  }
  
  watch(isOpen, (newVal) => {
    if (newVal) {
      nextTick(updateHeight);
    } else {
      updateHeight();
    }
  });
  
  onMounted(() => {
    updateHeight();
  });
  ```
  
  3. Update template to bind height style:
  
  ```vue
  <div 
    class="reliquary-slot__body-wrapper"
    :style="{ height: computedHeight }"
    :class="{ 'reliquary-slot__body-wrapper--open': isOpen }"
  >
    <div ref="bodyRef" v-show="isOpen" class="reliquary-slot__body">
      <slot>...</slot>
    </div>
  </div>
  ```
</action>
